    Logger

    Fork of the Logger class from the stdlib but without the deprecation notices.

    Installation

    1. Add the dependency to your shard.yml:

    `yaml dependencies:

     logger:
   github: 84codes/logger.cr

    `

    1. Run shards install

    Usage

    require "logger"
